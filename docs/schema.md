### 1. Kolekcja: `Filmy`
| Pole | Typ | Opis | Przykład |
| :--- | :--- | :--- | :--- |
| `tytuł` | **String** | Unikalna nazwa filmu. | `"Titanic"` |
| `data_premiery` | **Date** | Data wejścia filmu do kin. | `ISODate("1997-12-19")` |
| `gatunki` | **Array** | Lista gatunków danego filmu. | `["dramat", "romans"]` |
| `reżyser` | **String** | Imię i nazwisko reżysera. | `"James Cameron"` |
| `aktorzy` | **Array** | Lista aktorów danego filmu. | `["Kate Winslet", "Leonardo DiCaprio", "Billy Zane", "Kathy Bates"]` |
| `typDostępu` | **String** | Klasa filmu. | `"Premium"` |
| `licznikWyświetleń` | **Int** | Liczba oglądalności filmu. | `42567842` |
